Play:
https://siwoo-piano.vercel.app/

- ## 설명서

  - Play 버튼을 누를때 설정되있는 곡이 자동적으로 연주
  - 마우스 혹은 키보드를 사용하여 연주 가능

- ## 제작한 이유

  - 집에 피아노가 있었지만 너무 오래되어 소리를 먹는 키가 있어 만들어보고 싶었고
    악기를 전공하거나 따로 전문적으로 못하지만 직접 만들어서 쓰고 싶을때 쓸 수 있게 개발해보고 싶었기 때문이다.
